	- Mybatis-Config 설정
- 회원 공통 서비스
	- 회원정보 수정
	- 회원 탈퇴
- 배달원 서비스
	- 세탁물 수거
	- 배달진행
	- 배달완료

	- HTML CSS 등 전체적 UI/UX 개선 및 레이아웃 설정
- 고객 서비스
	- 예약 신청
	- 예약상태 변경
	- 이용 안내
	- 예약 수정
	- 예약 취소

	- PR 처리
	- 협업을 위한 STS, VSCode 의 자동 줄맞춤 통일
- 관리자 서비스
	- 세탁물 검수 진행
